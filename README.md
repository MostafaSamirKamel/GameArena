# 🎮 GameArena

A modern, dark-themed portal for gamers. The website serves as a landing page for a gaming community, highlighting trending streams, game releases, news, and community forums.

## 🚀 Features
- **Premium Dark UI**: Built with modern game-themed accents.
- **Latest News Grid**: Keep gamers updated with card layouts.
- **Stream/Video Carousel**: Mock area to watch trending gaming streams.
- **Interactive Forum Structure**: Mock UI representing community threads.

## 🛠️ Tech Stack
- **Frontend Core**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Custom gaming-style Webfonts
